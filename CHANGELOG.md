# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-10

### Added
- 6 种事件类型：done / confirm / perm / alert / daily / thinking
- Edge TTS 在线神经网络语音（默认 云希 温柔男声）
- Windows SAPI 离线 TTS 引擎（零依赖）
- winsound 电子音引擎（零依赖）
- 8 种中文语音（4 男声 + 4 女声）
- 音频缓存机制（首次在线生成，之后秒播）
- 支持重复播放和间隔设置
- JSON 格式输出
- WorkBuddy 技能包（双击安装）
- 通用命令行接口（适配所有 AI Agent 平台）
